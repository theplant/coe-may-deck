# May CoE Training Deck

[Slidev](https://sli.dev/) source for the May training: **AI Escapes the Chatbox / AI 走出聊天框**.

## Run locally

```bash
cd slides
npm install
npm run dev
```

Slidev opens at <http://localhost:3030>. Press `f` for fullscreen, `o` for overview, `?` for keybindings.

## Export to PDF

```bash
npm run export
```

## Structure

- `slides.md` — single-file deck source (Slidev convention)
- See `../.scratch/may-training/outline.md` for the full running order, timing, and demo plan.
